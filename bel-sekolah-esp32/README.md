# bel-sekolah-esp32

## Project Overview
This project is intended to build a web-based bell system using the ESP32 platform.

## Features
- Web-based control.
- Integration with the ESP32 hardware for bell functionality.

## Getting Started
### Prerequisites
- ESP32 Development Board
- Arduino IDE or PlatformIO

### Installation
1. Clone the repository.
2. Open the project in your IDE.
3. Configure your Wi-Fi credentials in the code.

### Usage
- Follow the documentation to set up the server on the ESP32.
